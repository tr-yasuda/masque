# Security Policy

## Supported versions

This project is in early development. Only the latest commit on the main
branch is supported.

## Reporting a vulnerability

If you discover a security issue, please do not open a public issue. Instead,
use one of the following private channels:

Open a draft [GitHub Security Advisory](https://github.com/tr-yasuda/masque/security/advisories/new)
in this repository.

Please include as much detail as possible, such as:

- Affected files or features.
- Steps to reproduce.
- Potential impact.
- Suggested mitigation, if any.

We will acknowledge receipt as soon as possible and keep you informed of our
progress.

## Scope

Because this is a research and proof-of-concept project, it is not intended
for production use or exposure to untrusted networks. Security reviews are
valuable but should be understood in that context.
